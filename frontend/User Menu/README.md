# User Menu Application

This is the customer-facing menu application that appears when users scan the QR code at their table.

## Features
- Browse restaurant menu
- Search for dishes
- Filter by category
- Add items to cart
- Place orders

## Running the App
```bash
npm install
npm run dev
```

The app will run on `http://localhost:3000`

## Routes
- `/` - Main menu page (accessible after QR code scan)
